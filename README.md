# E-Sim Store - Toko E-Sim Online

## Project Overview
E-Sim Store is an online platform that provides users with digital SIM cards (E-SIMs) for seamless connectivity around the globe. With our E-SIM technology, users can switch operators easily and enjoy instant activation without the need for physical SIM cards. The platform showcases various data packages with clear descriptions and enables users to purchase through popular messaging services.

## Installation
To run the E-Sim Store locally, follow these steps:

1. **Download or Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/e-sim-store.git
   cd e-sim-store
   ```

2. **Open the Project**
   Open `index.html` in your preferred web browser. No additional setup is required since this is a static website.

## Usage
Once the project is opened in a browser, you can navigate through the site using the menu at the top:

- **Home**: Overview of the E-SIM benefits.
- **Produk**: Displays the available E-SIM packages for purchase.
- **Kontak**: Contact information for customer inquiries.

You can initiate a purchase by clicking on the "Beli Sekarang" buttons for each product, which will redirect you to Telegram to complete the order.

## Features
- **Global Connectivity**: Access E-SIM services in over 170 countries.
- **Instant Activation**: Activate your E-SIM immediately upon purchase.
- **High Security**: Each E-SIM comes with advanced encryption to protect your data.
- **Responsive Design**: The site is designed to be mobile-friendly, allowing for easy access on any device.

## Dependencies
This project utilizes the following external libraries:
- **Font Awesome** for icons: [Font Awesome CDN](https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css)

No additional backend or server-side dependencies are required.
